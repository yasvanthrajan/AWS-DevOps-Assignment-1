\# AWS DevOps Engineer Internship - Assignment 1



\## Objective



Deploy a simple HTML website on an AWS EC2 Ubuntu instance using Nginx.



\---



\## AWS Services Used



\- Amazon EC2

\- Security Groups



\---



\## Linux Commands Used



```bash

sudo apt update

sudo apt install nginx -y

sudo systemctl status nginx

sudo systemctl restart nginx

df -h

free -h

ps -ef

```



\---



\## EC2 Setup Steps



1\. Launched an Ubuntu EC2 instance.

2\. Created a Security Group.

3\. Allowed SSH (22) and HTTP (80).

4\. Connected using SSH through Git Bash.

5\. Installed Nginx.

6\. Hosted a custom HTML page.



\---



\## Website



The website displays:



\- Name

\- College

\- Branch

\- Email

\- Current Date



\---



\## Learning Outcome



\- Created and configured an AWS EC2 instance.

\- Learned Linux commands.

\- Installed and managed the Nginx web server.

\- Hosted a static website.

\- Used GitHub for version control.
